## Deployment

1. Place `index.html` in the root of your web host (GitHub Pages, Netlify, etc.).
2. Ensure the **book cover image** is stored locally (or replace the `src` with your hosted image link).
3. Verify your Stripe link in all CTA buttons.
4. Optional: Enable HTTPS to prevent “Not Secure” warnings.
